# Sales & Customer Analytics Dashboard

## Project Overview

This project is an interactive Business Intelligence dashboard built using Microsoft Power BI.

The dashboard provides insights into:

- Sales Performance
- Customer Analytics
- Product Analysis
- YoY / QoQ / MoM Sales Trends
- Executive KPI Reporting

## Tools & Technologies

- Power BI
- DAX
- Power Query
- SQL
- Data Modeling
- Data Visualization

## Features

- Interactive Filters & Slicers
- Drill Through Reporting
- KPI Cards
- Time Intelligence Calculations
- Customer Segmentation
- Product Performance Analysis
